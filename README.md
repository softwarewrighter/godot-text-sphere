# Godot Text Sphere

A Godot 4.5 demo that displays 3D extruded text arranged in a rotating circle around a sphere.

**[Live Demo](https://softwarewrighter.github.io/godot-text-sphere/?ts=1765818050415)**  <also: [YouTube Comparison Video](https://www.youtube.com/watch?v=B0z5Plhw9lI&t=22s)   >

![Demo Screenshot](images/demo.png?ts=1734036349795)  

## Related Projects

- [three-text-sphere](https://github.com/softwarewrighter/three-text-sphere) - A similar rotating text sphere demo using Three.js instead of Godot

## Quick Start

```bash
cd godot_rotating_text_demo
./run.sh
```

## Scripts

| Script | Description |
|--------|-------------|
| `setup_godot_rotating_text_demo.sh` | Regenerates the entire project from scratch |
| `godot_rotating_text_demo/run.sh` | Runs the project in Godot |
| `godot_rotating_text_demo/export_mac.sh` | Exports a macOS .app bundle to `build/` |
| `godot_rotating_text_demo/export_web.sh` | Exports a web build to `build/web/` |

All scripts auto-detect Godot installation. Set `GODOT_BIN` environment variable to override.
